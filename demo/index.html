HOW IT WORKS


I'll be the first to admit, S-S isn't particularly complicated or clever or new. But that's exactly the point. It doesn't take that much extra effort to make the web a safer place, so we developers should do whatever we can to help, no matter how small.

That's why I wanted to make S-S. And here's how it protects your privacy.

1) Proxy server calls

If you want to know how many times a given link has been Shared/Tweeted, you can call the Facebook/Twitter APIs like so. (click links for example API calls)

Twitter: http://urls.api.twitter.com/1/urls/count.json?url={{URL}}
Facebook: http://graph.facebook.com/?id={{URL}}

S-S protects your privacy by calling these APIs through a proxy server. This way, they can't even log your IP address! By default, S-S uses JSONProxy. (open source, so you can run your own JSONProxy) If you want, you can easily customize S-S to use a different CORS proxy:

2) Static share links

These static links go to sharing pages for Twitter/Facebook. Just a simple hyperlink. No external iframe'd buttons. No Javascript libraries. No way it can track you just by simply being on the web page.

Twitter: https://twitter.com/share?url={{link}}&text={{text}}
Facebook: https://www.facebook.com/sharer/sharer.php?u={{link}}&t={{text}}

If you don't care about Share Counts, you can avoid using S-S altogether and just use these links by themselves.


3) Being open & customizeable

Would you fork me? I'd fork me.

All of this would be pointless if you couldn't freely use, examine, and modify my code.
S-S is dedicated to the public domain using the Unlicense. With the code being open to the public, you can glance through my fairly short code (under 150 lines with whitespace and comments), and make sure it really does protect your privacy. Yup. Nothing suspicious there.

CONCLUSION


I love sharing information! But I should have control over what I share.

That control over your generosity is the difference between letting a friend crash at your place, and them breaking into your apartment... then raiding your fridge for cookies, leaving a personalized ad for Oreos, and exiting through a backdoor that they secretly installed in everyone's houses.

S-S isn't going to save privacy. But it's a simple way to make our web slightly safer from spies. Every simple gesture counts
